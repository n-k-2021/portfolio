研究室ホームページ

所属研究室のウェブサイト

制作期間

2020年7月～約3ヶ月間

制作担当

プログラミング / デザイン

使用スキル

PHP / SQL / Javascript

About

所属研究室のスマホ対応ホームページです。以前の研究室ホームページはスマホ対応しておらず、 PCよりスマホを使う高校生及び大学1～３年生に研究室を知ってもらうために作成しました。 お知らせの投稿や、記載情報の変更をソースコードを直接編集せずにWeb上で行うことができます。

「言語ごとの開発」

開発言語が複数必要であったため、各言語ごとにグループで開発しました。

「デザインの決定」

サイトデザインを複数案から選定しました。

「サイトの完成」

グループごとに作成したプログラムやデザインを組み合わせ、 サイトを完成させました。

Role

このプロジェクトではリーダーを務めました。プロジェクトでは複数のプログラミング言語の習得と、メンバー全員が意欲的に参加すること が難点でした。そこで私は、制作の前にメンバーの経験や将来やりたいことを参考に言語ごとにグループ分けをして勉強会を開催しました。 また、サイトデザインをコンペ形式で決定することにしました。これらの取り組みから、メンバーの意欲を保ちながら魅力的な ホームページを完成させることができました。
