KYONO

家族の会話をつくるアプリケーション

制作期間

2019年4月～約5ヶ月間

制作担当

企画 / UIデザイン / プログラミング

使用スキル

HTML / CSS / JS / mBaaS / Illustrator / Photoshop

Concept

家族の会話をつくるアプリケーションです。スマートフォンの操作で会話が減っていると感じる親をターゲットとしたアプリです。 このアプリは、家族の一人ひとりがその日起きた些細な出来事を写真に収め、その日限定で見ることができる スライドショーをつくります。スライドショーが家族の会話をするきっかけをつくることができます。

「コンセプト決定」

新しいアプリを提案するためにまず個人企画を行い、近いアイデア同士でグループをつくりました。 そして、ブレインストーミングやKJ法を用いてアプリのコンセプトを決定しました。

「アプリ制作」

決定したコンセプトをもとにアプリの機能や構成について検討し、プロトタイプを作成した後、アプリ制作に移りました。 私はグループのリーダーを努めつつ、主にUIデザインとプログラミング（会員登録と画像共有機能、スライドショーの実装）を行いました。

「提案」

開発したアプリの良さを伝えるために、アプリの画面やプロトタイプの素材等を使って魅力的なプレゼンシートと紹介動画を作成しました。

Award

オージス総研主催OGIS-RI Software Challenge Awardへ応募し、一次審査通過しました。
